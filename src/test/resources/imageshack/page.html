Liens permettant de partager cette image ImageShack créé des connections automatiques et un code pour vos images. En utilisant ce code, vous pouvez vous connecter à vos fichiers de nombreuses façons. Vous pouvez intégrer vos fichiers à des pages web ou les montrer à vos amis par le biais d'emails ou de messagerie instantanée. .

Pour utiliser ces liens ou codes, cliquez à l'intérieur de la boîte de code puis appuyez sur CTRL+C afin d'en copier le lien. Le lien du code est désormais enregistré dans le "presse-papier" de votre ordinateur et peut être collé nimporte où. Vous pouvez dès lors ouvrir l'emplacement choisi (Windows Messenger ou une boîte de messagerie) et y coller votre lien ou code.

"Lien"est la page de renvoi des images pour Imageshack et S'avère particulièrement efficace pour les partager via Email et la Messagerie Instantanée."Direct"est l'emplacement d'hébergement de l'image sur les serveurs d'ImageShacks.
